MRH_GetEvDataLibraryVersionMinor
================================
The MRH_GetEvDataLibraryVersionMinor function is used to obtain 
the minor library version.

Header
------
The function definition can be found in the following file:

.. code-block:: c

    #include <libmrhevdata/MRH_EvDataRevision.h>


Syntax
------
.. code-block:: c

    int MRH_GetEvDataLibraryVersionMinor(void);


Parameters
----------
None.

Return Value
------------
The minor library version defined by 
:doc:`../../../Macros/libmrhevdata_Directory/MRH_EvDataRevision/MRH_EV_DATA_LIB_VERSION_MINOR`.

Remarks
-------
None.

-------------
None.
