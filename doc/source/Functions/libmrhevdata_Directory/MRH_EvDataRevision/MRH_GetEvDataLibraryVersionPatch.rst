MRH_GetEvDataLibraryVersionPatch
================================
The MRH_GetEvDataLibraryVersionPatch function is used to obtain 
the library patch version.

Header
------
The function definition can be found in the following file:

.. code-block:: c

    #include <libmrhevdata/MRH_EvDataRevision.h>


Syntax
------
.. code-block:: c

    int MRH_GetEvDataLibraryVersionPatch(void);


Parameters
----------
None.

Return Value
------------
The library patch version defined by 
:doc:`../../../Macros/libmrhevdata_Directory/MRH_EvDataRevision/MRH_EV_DATA_LIB_VERSION_PATCH`.

Remarks
-------
None.
